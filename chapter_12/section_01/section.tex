\section{Soil Moisture Sensing} \label{sec:}

\begin{center}
    {\begin{minipage} [c] {0.55\textwidth}

        Soil moisture sensing is done using resistive soil moisture sensors.
        The soil moisture content then can be estimated by appropriately
        scaling the voltage read from the sensors. The multiplexer board is
        used to interface between the sensors and the \esp. See figure
        \ref{fig:soilMoistureSensor} for the sensor used.

        Specification:

        \begin{itemize}
            \item \textbf{Operating voltage:} $3.3\si{V}$ to $5\si{V}$.
        \end{itemize}
